Welcome to my phase 3 end project for the Simplilearn Full Stack Developer Course.
The Project consists of 3 main Services:
  • One micro service for the Thymeleaf frontend and for accessing and storing the Cab bookings
  • One micro service for calculating the cab fare and for listing the possible routes
  • A Eureka server to connect the microservices
The BookingService micro service accesses the CabFare micro service via a Rest API
